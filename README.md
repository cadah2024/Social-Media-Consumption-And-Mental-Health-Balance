# Overview
This project analyzes how social media usage affects mental health using a dataset of 500 individuals. It explores key variables such as daily screen time, sleep quality, stress levels, exercise frequency, and preferred social media platforms. The goal is to identify patterns and relationships that show how digital habits influence overall well-being. The analysis includes data cleaning, feature engineering, visualization in Power BI, and insights that help explain the impact of social media on happiness, stress, and lifestyle balance.
